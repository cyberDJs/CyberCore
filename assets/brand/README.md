# CyberCore Brand Assets

Status: v0.1.0 foundation release

These assets are deterministic SVG files intended for repository, documentation,
and release-page use.

## Files

| File | Use |
|---|---|
| `cybercore-mark.svg` | Compact icon mark |
| `cybercore-logo.svg` | Logo lockup for README and docs |
| `cybercore-social-preview.svg` | GitHub social preview or release header |

## Visual direction

The v0.1 visual identity follows the design-system direction:

- minimalist cyberpunk;
- technical dashboard clarity;
- high contrast without decorative noise;
- color as semantic signal.

## Rules

- Do not add secrets, screenshots of private infrastructure, customer names, or production topology to public brand assets.
- Prefer SVG for repo-native identity assets so changes remain reviewable.
- Use generated bitmap imagery only when the asset depends on illustration, texture, or photorealistic composition.
