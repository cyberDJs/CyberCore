# CyberCore

**AI-first open infrastructure platform powering the CyberDJS ecosystem with automation, observability, security, documentation and self-healing operations.**

CyberCore is the control-plane project for the Eimy Herrer / CyberDJS infrastructure ecosystem. The first deployment target is InterServer: shared hosting, VPS, DirectAdmin, mail, DNS, WordPress, Nextcloud and related services.

See `roadmap.md` for sprint tracking and `docs/manual-actions.md` for manual actions.
